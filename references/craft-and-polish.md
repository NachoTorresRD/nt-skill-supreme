# NT-Design Engine: Visual Quality Audit & Craft Reference

*A complete operational manual for auditing, polishing, colorizing, and perfecting frontend craft.*

---

## 🔬 Visual QA Audit Protocol

When running a visual QA audit (`/design-audit`), systematically inspect:

1. **Hierarchy Check**: Is the primary focal point immediately obvious? Are secondary and tertiary actions visual subordinates?
2. **Spacing Cadence**: Are padding and margins uniform? Are inner card paddings consistently smaller than container margins?
3. **Contrast Balance**: Is text legible on all background image gradients or glass surfaces?
4. **Interaction Completeness**: Do buttons have default, hover, active, focus, disabled, and loading spinner states?
5. **Micro-Typography**: Are numbers tabular (`font-variant-numeric: tabular-nums`) in tables and counters? Are headings free of widow words?

## 🎛 The Craft Manual Commands

| Command | Objective | Action Taken |
|---|---|---|
| `adapt` | Fluid Layouts | Adapts static layouts for seamless mobile, tablet, and widescreen responsiveness. |
| `animate` | Motion Polish | Converts instant DOM state changes into smooth physics-based spring transitions. |
| `audit` | Codebase Scan | Performs a full visual hierarchy, accessibility, and slop inspection. |
| `bolder` | Visual Impact | Enhances visual contrast, key typography weights, and accent focal points. |
| `colorize` | Palette Harmony | Replaces generic default colors with sophisticated HSL adaptive palettes. |
| `critique` | UX Review | Evaluates usability friction, cognitive load, and visual hierarchy. |
| `delight` | Micro-Surprises | Introduces contextual feedback, subtle celebratory state triggers, and confetti/particles. |
| `polish` | Final Cleanliness | Cleans alignment, normalizes radii, refines border opacities, and standardizes shadows. |
| `quieter` | Noise Reduction | Softens visually loud elements, reduces unnecessary borders, and cleans background clutters. |
| `typeset` | Typography Spec | Establishes strict modular scale font sizes, line heights, tracking, and font stacks. |
